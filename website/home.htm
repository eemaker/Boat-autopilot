CAPTAIN: Autopilot control center

CAPTAIN is an abbreviation of "Captain AutoPiloT And Intelligent Navigation". This platform makes it possible to control and issue navigation commands to a boat from an intuitive interface.

Point to point

This function enables you to sail from your current position to another in a straight line.

Your destination can be selected either on the map or using coordinates.

Coverage

The cover function enables you to cover an rectangular area.

Status

The status page displays information about your location, connection, and boat motors for diagnostic purposes.

Edit Parameters

Edit parameters allows you to create, save, delete, edit and select PID profiles for the boat.
